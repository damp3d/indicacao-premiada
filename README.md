# Indicação Premiada

Sistema web para gestão de indicação premiada com:

- super admin para gerenciar lojas
- painel individual por loja
- formulário público por loja
- cadastro de indicadores
- controle de benefícios e resgates
- Firebase para autenticação e banco de dados

## Rodar localmente

Pré-requisitos:

- Node.js instalado

Passos:

1. instale as dependências com `npm install`
2. configure o `.env` e o `.env.local`
3. rode `npm run dev`

Aplicação local:

- `http://localhost:3000`
